# Greet Skill

## Purpose
A simple demonstration skill. When the user asks Argus to greet them or say hello, this skill provides the framework for a warm greeting.

## Usage
- Trigger: User asks for a greeting or says hello.
- Action: Respond with a warm, personalized greeting using the user's name from persona.

## Scripts
- `greet.js` — Outputs a greeting message to stdout.
